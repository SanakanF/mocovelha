# MocoVelha

Projeto inicial do jogo da velha com backend em FastAPI e frontend estático.

## Estrutura

```
MocoVelha/
  backend/
    ai/
      qlearning.py
      game.py
    api/
      endpoints.py
      schemas.py
    main.py
    requirements.txt
    README_BACKEND.md

  frontend/
    index.html
    script.js
    style.css
    README_FRONTEND.md

  .gitignore
  README.md
```

## Como começar

- Consulte `backend/README_BACKEND.md` para executar o servidor FastAPI.
- Abra `frontend/index.html` em um navegador para visualizar a interface inicial do jogo da velha.
